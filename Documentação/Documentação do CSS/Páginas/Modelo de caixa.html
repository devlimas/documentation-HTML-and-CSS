Para o uso de elementos flutuantes, vou mostrar usando divs com cores:

Olá div1


Viu só, com o uso do elemento de estilização float no css, a tag "div2" agora está em cima da "div 1" mas porque isso acontence sendo que cada div é um bloco que ocupa a linha inteira ?


Simples, porque esta div estar float(flutuando), ela se tornou um elemento flutuante, logo ela não está em nenhuma linha, ela está apenas flutuando pela página podendo ocupar e ficar em cima de qualquer outra coisa, usando float com outras tags como width e height, além do padding, conseguimos fazer muitas coisas interresantes sem interferir na página de forma direta
