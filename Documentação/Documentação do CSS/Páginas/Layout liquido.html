Explicação do layout liquido

Para deixamos um layout responsivo, deixamos de usar px no width de qualquer div, PRINCIPALMENTE NA DIV PRINCIPAL DA PÁGINA, é dessa div que teremos as medidas das outras, usando uma regra de 3, se a div principal tem 720px e trocamos por 85% de width, vamos usar a sua medida em px para fazer uma regra de 3 com as próximas divs assim fazendo elas respeitar o tamanho da div principal, exemplo: a div lobby tem 520px e outra div conteudo de 180px, somando os 2 tem 700px, lembrando que a div principal tem 720px, os outros 20px são o espaçamento, então se eu pegar a div lobby e multiplicar por 100, vai dar 52000, depois dividimos pelo total, 52000/700, dando 74,28. O calculo vai ser o mesmo com a div conteudo
