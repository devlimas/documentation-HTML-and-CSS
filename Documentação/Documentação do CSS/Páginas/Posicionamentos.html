Viu, mas não mudou nada certo ? de fato, o posicionamento estático ou "static" é o posicionamento padrão e por isso não notamos diferenças

Veja, agora este h1 e a próxima tag "p estão com o posicionamento relativo, ou seja "relative"

Viu, a posição relativa mudou o posicionamento do "h1" e do "p" lembre que para usar o relative, temos que usar junto com ele elementos como left, right, top e bottom, usar apenas o relative não vai mudar o posicionamento das tags

Agora vamos para o absoluto, este h1 e a próxima tag "p estão com o posicionamento relativo, ou seja "absoluto"

Você ver como o "absolute" é extremamente agressivo como elemento flutuante, para pode usar um posicionamento relativo dentro de div, assim tudo dentro dela mesmo com a posição "absolute", ele vai respeitar
